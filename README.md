# SQL Data Analysis Projects

A collection of SQL projects, database setup files, and advanced SQL practice problems designed to improve SQL querying and data analysis skills.

## 📂 Project Structure

```
SQL-Projects/
│
├── 📁 project_sql
│   ├── 1_top_paying_jobs.sql
│   ├── 2_top_paying_job_skills.sql
│   ├── 3_top_demanded_skills.sql
│   ├── 4_top_paying_skills.sql
│   └── 5_optimal_skills.sql
│
├── 📁 sql_load
│   ├── Database loading scripts
│   └── Instructions to import the dataset
│
└── 📁 Advanced_sql
    └── SQL practice problems and it's solutions
```

---

## 📁 Folder Description

### 1. `project_sql`
Contains real-world SQL data analysis projects based on a job market dataset. Each SQL file answers a specific business question using SQL concepts such as:

- Filtering
- Joins
- Aggregations
- CTEs
- Window Functions
- Ordering & Ranking
- Subqueries

---

### 2. `sql_load`
Contains everything required to set up the database before running the project queries.

This folder includes:

- Database import instructions
- SQL scripts for loading the dataset
- Initial setup files

---

### 3. `Advanced_sql`
A collection of SQL practice problems along with their solutions.

Topics include:

- Joins
- GROUP BY
- HAVING
- CTEs
- Window Functions
- Subqueries
- Ranking Functions
- Case Statements
- Advanced Query Techniques

---

## 🛠 Requirements

- PostgreSQL (recommended)
- pgAdmin or any SQL client
- Basic SQL knowledge

---

## 🚀 Getting Started

1. Open the **sql_load** folder.
2. Import the database by following the provided instructions.
3. Explore the queries inside **project_sql**.
4. Practice additional concepts using **Advanced_sql**.
